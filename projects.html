As a student at Ithaca College I worked on a number of other software projects:

Guessong is a multiplayer web game I contributed to. Random songs are played and users must guess the name of the song before the other players! We wrote the backend in Python/Flask and used JQuery and Bootstrap on the frontend. Realtime client-server communication was handled with SocketIO. The audio for the songs was retrieved using the ItunesAPI.

My Compiler is a project I started in the summer of 2018 and got academic credit for the following semester. I implemented most of the features for a compiler for a simple C-like language. This was a huge undertaking but I learned a lot and it was really fun.
